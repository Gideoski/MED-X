# **App Name**: Med-X Hub

## Core Features:

- User Authentication: Secure account creation, login, and password management.
- E-Book Display (100lvl & 200lvl): Display free and premium e-books based on user level.
- Search Functionality: Enable users to search for specific materials within each level.
- Premium Subscription: Manage premium content access based on subscription status using Paystack for payments.
- Content Upload (Admin/Creators): Allow verified creators and admins to upload and manage e-book content for different levels.
- Feedback System: Collect user feedback via a form that sends directly to the admin.
- AI-Powered Help Bot: A chatbot tool providing instant answers and guidance to user queries. The tool uses its judgement to respond to queries appropriately.

## Style Guidelines:

- Primary color: Deep red (#B71C1C) to represent medical themes.
- Background color: Light gray (#EEEEEE) for a clean, readable interface.
- Accent color: Black (#000000) for contrasts and CTAs.
- Body and headline font: 'Inter' sans-serif, ensuring a modern and clean look.
- Use simple, clear icons in black to represent different services and navigation items.
- Maintain a clean, responsive layout with clear sections for each service. Utilize a hamburger menu for navigation on smaller screens.
- Subtle transitions and loading animations to enhance user experience without being distracting.
